## The Oracle Pattern คืออะไร

หนังสือเล่มนี้อธิบายการออกแบบ **Oracle System**: AI agent ที่มี identity, memory, soul portable ใน repo และทำงานเป็นทีมผ่าน **MAW Engine** หรือ Multi-Agent Workflow.

แกนของหนังสือคือแนวคิดว่า Oracle ไม่ใช่แค่ bot ตอบข้อความ แต่เป็นระบบที่มี:

- **Code + Soul** — โค้ดทำงานร่วมกับ `ψ/` vault ที่เก็บตัวตน ความจำ และบทเรียน
- **MAW Engine** — orchestration layer สำหรับสั่งงาน AI หลายตัวแบบเป็นขั้นตอน
- **Team Charter** — จาก YAML ไปเป็น pipeline และทีม agent ที่ทำงานซ้ำได้
- **Cross-engine teams** — Claude, Codex และเครื่องมืออื่นทำงานร่วมกันได้
- **Soul Portable** — ตัวตนย้ายเครื่องหรือ repo ได้โดยไม่หายไปพร้อม runtime
